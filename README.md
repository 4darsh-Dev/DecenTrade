### Prerequisites

- Node.js (v16.0.0 or later)
- npm (v6.0.0 or later)
- MetaMask or similar Ethereum wallet

### Installation

1. Clone the repository
```bash
git clone https://github.com/4darsh-Dev/DecenTrade.git
```
2. Install dependencies
```bash
  cd DecenTrade
  npm install
```
3. Start the Development Server
```bash
  npm run dev
```
